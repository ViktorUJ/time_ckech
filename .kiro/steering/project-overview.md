# Обзор проекта: Сервис родительского контроля

## Описание

Windows-служба (Windows Service) на Go для ограничения использования компьютера ребёнком в развлекательных целях. Целевая платформа: Windows 10/11 x64.

## Технологический стек

- Язык: Go (1.22+)
- Платформа: Windows 10/11 64-bit
- Windows API: `golang.org/x/sys/windows`, `golang.org/x/sys/windows/svc`
- HTTP-клиент: стандартная библиотека `net/http` для загрузки конфигурации с GitHub
- HTTP-сервер: стандартная библиотека `net/http` для удалённого доступа к логам (только LAN)
- Логирование: Windows Event Log (`golang.org/x/sys/windows/svc/eventlog`)

## Структура проекта

```
cmd/
  service/         — точка входа Windows-службы
  installer/       — утилита установки/удаления службы
internal/
  config/          — менеджер конфигурации (загрузка с GitHub)
  monitor/         — мониторинг процессов
  browser/         — мониторинг браузеров
  scheduler/       — расписание и управление временными окнами
  enforcer/        — блокировка программ и уведомления
  sleepmode/       — управление режимом «Время сна»
  state/           — сохранение/восстановление состояния
  logger/          — обёртка над Windows Event Log и полное логирование
  httplog/         — HTTP-сервер для удалённого доступа к логам (LAN only)
```

## Конфигурационные файлы (GitHub)

- `allowed_apps.json` — список разрешённых программ
- `allowed_sites.json` — список разрешённых сайтов
- `schedule.json` — расписание временных окон, время сна и параметры логирования

## Ключевые принципы

- Сервис работает под учётной записью Local System
- Системные процессы Windows всегда исключаются из мониторинга
- Время развлекательного контента считается по wall clock (не суммарно по программам)
- Конфигурация обновляется каждые 5 минут с GitHub
- Состояние сохраняется на диск каждые 30 секунд
- Режим «Время сна» блокирует все программы, включая разрешённые
- Полное логирование и HTTP-доступ к логам включаются через конфигурацию на GitHub
- HTTP-сервер логов принимает подключения только из локальной сети
